1. logo.png 为网站logo, 尺寸 为 高 60px , 最大宽 201px , 请限制传入图片高,手机端在不同地方处理logo高,改动大

2. favicon.png为网站标签图标,尺寸 48* 48px

3. img目录为游戏入口图片及游戏详情图片
    img/*.png   => 首页入口图片，注意前缀， live-*.png 为真人 ， slot-*.png为电子， game-*.png 为体育和彩票
    img/game/*.png  => 游戏详情图片，注意前缀， LIVE-*.png 为真人 ， SLOT-*.png为电子， OTHER-*.png 为体育和彩票

4. 其他参数见keys.config.json,其中对字段有说明
